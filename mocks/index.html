What's not yet mocked (deliberately deferred): Write-Up + Acknowledge token flow, Solicit Feedback + Response token flow, Goals form, Meeting form, Quick Feedback form, PIP form, Settings/Configuration, Permissions Admin. (Network, School, and Team views no longer live in OP — they're owned by PeoplePoint.) All exist in the FirstLine code; the visual shape will follow the patterns set in the mocks here.

Aesthetic: calm river palette, deep navy hero, teal for PeoplePoint, slate-blue for ObservationPoint. Status colors deliberately muted — no alarmist red. Sidekick voice throughout. Cross-app switching lives in the suite top bar so the two products feel like one.

Schools: fictional — Horizon Public Schools (Horizon Middle, Cedar Grove, Pinecrest) — per the demo-not-FirstLine rule.
