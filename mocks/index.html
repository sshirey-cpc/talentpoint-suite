What's not yet mocked (deliberately deferred): Network drilldowns by category (observations, evals, action steps, fundamentals), Write-Up + Acknowledge token flow, Solicit Feedback + Response token flow, Goals form, Quick Feedback, Quick Meeting, PIP, Settings/Configuration, Permissions Admin. All exist in the FirstLine code; the visual shape will follow the patterns set in the mocks here.

Aesthetic: calm river palette, deep navy hero, teal for PeoplePoint, slate-blue for ObservationPoint. Status colors deliberately muted — no alarmist red. Sidekick voice throughout. Cross-app switching lives in the suite top bar so the two products feel like one.

Schools: fictional — Horizon Public Schools (Horizon Middle, Cedar Grove, Pinecrest) — per the demo-not-FirstLine rule.
